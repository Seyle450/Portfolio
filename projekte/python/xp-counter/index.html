XP-Counter

Sept. 2023 · Python

Erfahrungspunkte- und Level-Tracker mit fortlaufender Berechnung.

  • Python
Läuft direkt im Browser (Python via Pyodide) — Ein- und Ausgabe im Terminal.
main.pyPython
Total = 0
exp = 1000
Level = 52
average = 500
Minutes = 0
Hours = 0
while Level < 100:
    exp = 1000 * Level
    print ("Level", Level, "XP", exp)
    Total += exp
    Level += 1
    
print("")
print("Du brauchst", Total, "EXP um Level", Level, "zu erreichen.")
Minutes = Total / average
Hours = Minutes / 60
print("Das wären ungefähr", Hours,"Stunden, oder",Minutes,"Minuten.")
